Getting Started Guide

Welcome to the WSBSD Revamp Getting Started Guide! This guide will help you set up and start using WSBSD Revamp, the modern reimplementation of Windows Subsystem for BSD. Follow the steps below to get up and running with WSBSD Revamp on your Windows system.

System Requirements

Before installing WSBSD Revamp, ensure your system meets the following requirements:

  • Windows 11
  • At least 4 GB of RAM
  • 544 KB of disk space for installation files
  • 10 GB of free disk space available
  • Visual C++ Redistributable Runtime 2017-2026 (yes it is named like that, according to the Microsoft page)

Installation

Alright, let's first install Visual C++ Redistributable Runtime 2017-2026. (Skip this step if you already have it installed.) You can click here to download it. Click on the X64 Version and follow the installation instructions.

Then, download the latest installer from the Downloads Page.

Running the Installer

Once you have downloaded the WSBSD Revamp installer, follow these steps:

  1. Run the WSBSD Revamp installer (.exe file)
  2. On the first page, click Next
  3. On the second page, click Install
  4. Wait for the installation to complete. The installer will set up all available BSD distributions (OpenBSD, NetBSD, and FreeBSD)

First Use

After installation is complete:

  1. Click the Start Menu
  2. Search for one of the available BSD systems: OpenBSD, NetBSD, or FreeBSD
  3. Click on your preferred choice to launch the BSD environment

Basic Commands

Once you have launched a BSD environment, you can type help to see a list of available commands.

The cc command is the C compiler available in WSBSD Revamp. It currently supports compiling assembly files using NASM (Netwide Assembler) and LLVM's ld.ldd (linker) to produce executable files (a.out).

Example:

cc program.asm

This will compile your assembly file and create an executable output file.
