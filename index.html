Welcome to WSBSD Revamp - the modern reimplementation of Windows Subsystem for BSD. Experience native BSD environments on Windows with improved performance, compatibility, and architecture.

What is WSBSD Revamp?

WSBSD Revamp is the next generation of Windows Subsystem for BSD (WSL1 for BSD), written entirely in C++ for performance and efficiency. It features an ELF x86-64 Emulator to enable seamless execution of BSD binaries on Windows. Hosted on GitLab at Coolis1362/wsbsd-revamp, it represents a modernized approach to running BSD environments natively on Windows systems with enhanced compatibility, performance, and user experience.

Where is WSBSD Classic?

WSBSD Classic is the original implementation of Windows Subsystem for BSD. It started as Python scripts and Batch files in its early phases, but has since evolved and is now written in .NET Framework 4.7.2 C#. Hosted on GitHub at TTSConsulting/WSBSD, it laid the foundation for seamless BSD integration on Windows. WSBSD Revamp builds upon that legacy with a complete rewrite in C++ and an integrated emulator for improved performance and architecture.

Project Timeline

On August 11th, 2025, WSBSD Classic's last release (version 1.0.0.4) was published, with the final commit made on the same day. This marked the end of active development for the original project. However, on June 5th, 2026, the WSBSD Revamp repository was created and first committed on the same day.
